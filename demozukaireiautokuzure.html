テキストレイアウト崩れ対応デモサイト

このデモサイトでは、日本語から英語へのテキスト置換時によく発生するレイアウト崩れの問題と、それを解決するためのCSS技術を紹介します。各例で「JP/EN」ボタンをクリックすると、日本語と英語のテキストを切り替えることができます。

1. メニュー項目の問題

問題例 JP/EN

固定幅のメニュー項目で、英語テキストが長くなると折り返しやはみ出しが発生します。

ホーム
会社概要
製品情報
お問い合わせ
Home
Corporate Information
Products
Contact Us
.header-menu-item { width: 120px; /* 固定幅 */ }
解決策 JP/EN

最小幅を指定しつつ、white-spaceプロパティを使用してテキストの折り返しを防ぎます。

ホーム
会社概要
製品情報
お問い合わせ
Home
Corporate Information
Products
Contact Us
.header-menu-item { min-width: 120px; /* 最小幅の指定 */ white-space: nowrap; /* テキストの折り返しを防止 */ }

2. ボタン内のテキスト問題

問題例 JP/EN

固定幅のボタンで、英語テキストがはみ出す問題が発生します。

詳細を見る
View Details
.fixed-width-button { width: 120px; /* 固定幅 */ text-align: center; }
解決策 JP/EN

最小幅を指定しつつ、自動幅とパディングを使用して柔軟に対応します。

詳細を見る
View Details
.auto-width-button { min-width: 120px; /* 最小幅 */ width: auto; /* 自動幅 */ padding: 8px 15px; /* 余白の指定 */ }

3. 単語途中での改行問題

問題例 JP/EN

英単語が長い場合、固定幅要素内で単語が切れてしまいます。

環境問題について詳細な情報を提供しています。
We provide comprehensive information about environmental sustainability issues.
.long-word { width: 150px; /* 固定幅 */ }
解決策1: word-break JP/EN

word-break: break-all を使用すると、単語途中でも強制的に改行します。

環境問題について詳細な情報を提供しています。
We provide comprehensive information about environmental sustainability issues.
.break-all { word-break: break-all; /* 単語途中でも改行 */ }
解決策2: word-wrap JP/EN

word-wrap: break-word を使用すると、単語単位で改行し、必要な場合のみ単語内で改行します。

環境問題について詳細な情報を提供しています。
We provide comprehensive information about environmental sustainability issues.
.break-word { word-wrap: break-word; /* 必要な場合のみ単語内で改行 */ }

4. テキスト切り捨てとハイフネーション

解決策3: テキスト切り捨て JP/EN

テキストを折り返さず、はみ出た部分を省略記号で表示します。

環境問題について詳細な情報を提供しています。
We provide comprehensive information about environmental sustainability issues.
.truncate { white-space: nowrap; /* 折り返しを禁止 */ overflow: hidden; /* はみ出た部分を隠す */ text-overflow: ellipsis; /* 省略記号を表示 */ }
解決策4: ハイフネーション JP/EN

長い単語をハイフンで区切って改行します（ブラウザ対応に注意）。

環境問題について詳細な情報を提供しています。
We provide comprehensive information about environmental sustainability issues.
.hyphenate { hyphens: auto; /* 自動ハイフネーション */ -webkit-hyphens: auto; -ms-hyphens: auto; }

5. レスポンシブ対応テクニック

レスポンシブコンテナの活用

flexboxやgridを使用して、柔軟なレイアウトを実現します。

.container { display: flex; flex-wrap: wrap; } @media (max-width: 768px) { .container { flex-direction: column; } }

ビューポート単位の活用

固定幅ではなく、vwやvhなどのビューポート単位を使用します。

.responsive-element { width: calc(100% - 20px); max-width: 500px; font-size: calc(14px + 0.5vw); }

このデモサイトは、日本語から英語へのウェブサイト翻訳時に発生するレイアウト問題の対応方法を示すために作成されました。

実際のプロジェクトでは、これらの技術を組み合わせて最適な解決策を見つけることが重要です。
